Trattoria Sora Cencia v3

Per vedere il sito correttamente, devi usare un server locale.
Ecco i modi più semplici:

🔧 Metodo 1 - Windows (più semplice)

  1. Fai doppio clic su AVVIA_SERVER.bat
  2. Attendi che appaia "Serving HTTP on :: port 8000"
  3. Clicca qui sotto per aprire il sito
🌐 Apri il sito IT 🌐 Open EN site

🔧 Metodo 2 - Mac/Linux

  1. Apri il Terminale nella cartella del sito
  2. Scrivi: python3 -m http.server 8000
  3. Clicca qui sotto per aprire il sito
🌐 Apri il sito IT 🌐 Open EN site

🔧 Metodo 3 - VS Code

  1. Apri la cartella del sito in VS Code
  2. Installa l'estensione "Live Server"
  3. Clicca "Go Live" in basso a destra

📄 Apri index.html (senza grafica)

Nota: aprire index.html direttamente mostrerà il sito senza CSS/immagini
a causa delle restrizioni di sicurezza del browser.
